Practica inicial de CSS

Estructura basica de CSS

1/ CSS significa "Hojas de Estilo en Cascada"

2/ se utiliza para definir el estilo y la apariencia visual de un documento HTML y el que lo interpreta es el navegador web

3 y 4/ cada usuario define las reglas que especifican los grupos de estilos que van a aplicarse a elementos particulares o grupos de elementos de la página web. la regla se abre con un selector. Este selecciona el elemento HTML que vamos a diseñar. En este caso, diseñaremos encabezados de nivel uno

1.5/ Un selector nos permitirá aplicar estilos a elementos específicos de HTML. De esta forma es posible ser preciso al momento de aplicar diseño a un sitio web.

1.6/ Selector de ID Este tipo de selector es igual al de clase, aunque lo aplicaremos a un único elemento HTML de nuestra página. Esto se debe a que el atributo ID de HTML identifica a un elemento HTML por medio de un nombre

Selector de clases Los selectores de clases comienzan siempre por un . (punto). Los clientes web (navegadores) identifican este punto y lo distinguen del resto de los selectores.

Selector universal Nos permite aplicar estilos a todos los elementos HTML de un documento y se indica mediante el *.

Selector por etiquetas Aplicamos propiedades CSS a una etiqueta. La misma se identifica sin utilizar >.

Selector descendente Con este selector seleccionamos un elemento que está dentro de otro del mismo tipo. Estos seectores siempre están formados por dos o más selectores separados entre si. El último selector será el elemento HTML al que se le aplicará las propiedades de CSS.

Selector descendente Con este selector seleccionamos un elemento que está dentro de otro del mismo tipo. Estos seectores siempre están formados por dos o más selectores separados entre si. El último selector será el elemento HTML al que se le aplicará las propiedades de CSS.

1.7/ el css funciona:

  1. el navegador carga el HTML
  2. se transforma el html en dom
  3. el navegador busca los archivos vinculados al documento
  4. el navegador analiza el css para ordenarlo segun sus reglas y selectores
  5. el arbol de renderizacion dee la estructura en la que se debe aparecer la pagina lueg9o analizar sus reglas
  6. ultimo la pagina se muestra en pantalla

1.8/ las 3 maneras son: 1-Utilizando un atributo "style" 2-Utilizando la etiqueta "style" 3-Utilizando un archivo propio de css,es decir, con extension .css

1.9/ Si son iguales en el html pero son diferentes en el css,es decir que cambia el estilo. https://csszengarden.com/196/ https://csszengarden.com/195/ No son los mismos ya que cada uno tiene un estilo propio. Si,el mismo html puede verse distinto gracias al css, el cual cambia el estilo de la pagina.
